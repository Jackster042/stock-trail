# Stock Trail - Warp Workflows

---

## Development

### Start Development Server
Start the Next.js development server with Turbopack

```
npm run dev
```

---

### Build Production
Build the application for production

```
npm run build
```

---

### Start Production Server
Start the production server

```
npm run start
```

---

## Code Quality

### Run Linter
Check code for linting errors

```
npm run lint
```

---

### Type Check
Run TypeScript type checking

```
npx tsc --noEmit
```

---

## Database

### Test Database Connection
Test MongoDB database connection

```
npm run test:db
```

---

## Package Management

### Install Dependencies
Install all project dependencies

```
npm install
```

---

### Update Dependencies
Update all dependencies to latest versions

```
npm update
```

---

### Clean Install
Clean install of dependencies (removes node_modules first)

```
Remove-Item -Recurse -Force node_modules; npm install
```

---

## Git Operations

### Check Status
View current git status

```
git status
```

---

### Commit Changes
Stage all changes and commit

```
git add .; git commit -m "{{commit_message}}"
```

---

### Push Changes
Push changes to remote repository

```
git push
```

---

### Pull Latest
Pull latest changes from remote

```
git pull
```

---

## Project Info

### View Project Structure
List project directories and files

```
Get-ChildItem -Recurse -Depth 2 -Directory | Select-Object FullName
```

---

### Check Node Version
Display current Node.js version

```
node --version
```

---

### Check NPM Version
Display current npm version

```
npm --version
```
